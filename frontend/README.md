# Frontend

Interface HTML statique pour Barisense.

## Contenu
- Page unique en HTML/CSS générée avec Vite sans dépendances React.
- Données factices intégrées côté client (cafés, shots, verdicts).
- Aucune logique de formulaire interactive, seulement un rendu immédiat.

## Démarrage
```bash
cd frontend
npm install
npm run dev
```

## Construction
```bash
npm run build
```
