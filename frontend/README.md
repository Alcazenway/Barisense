# Frontend

Interface utilisateur responsive pour Barisense.

## Objectifs
- Formulaires web pour les cafés, shots et dégustations.
- Visualisation des classements et verdicts.
- Expérience mobile-first compatible PC, tablette et smartphone.

## Pistes techniques
- Framework SPA (React, Vue, Svelte ou équivalent).
- Gestion d’état pour séparer données techniques, sensoriel et décisions.
- Appels API au backend pour récupérer et envoyer les données.

## À faire
- Initialiser le projet front (build, lint, tests).
- Définir la structure de pages / routes et les composants de formulaires.
- Mettre en place les vues de classement et de verdicts.
